
Terraform - opensource проект от HashiCorp, является инструментом для создания ***Инфраструктуры в Коде (Infrastructure as a Code)***

Работает с такими cloud:
* AWS
* Google Cloud Platform
* Microsoft Azure
* Digital Ocean

Синтаксис кода ***Hashicorp Configuration Language (HCL)***,  не требует компиляции, написан на GO.

GitRepo - https://github.com/hashicorp/terraform
