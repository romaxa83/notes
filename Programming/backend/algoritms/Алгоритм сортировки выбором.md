-  –ö–æ–≥–¥–∞ –Ω–∞—à–ª–∏ —Å–∞–º—ã–π –º–∞–ª–µ–Ω—å–∫–∏–π —ç–ª–µ–º–µ–Ω—Ç –≤ —Ç–µ–∫—É—â–µ–º –æ—Å—Ç–∞—Ç–∫–µ, –º–µ–Ω—è–µ–º –µ–≥–æ –º–µ—Å—Ç–∞–º–∏ —Å –ø–µ—Ä–≤—ã–º.
-  –¢–µ–ø–µ—Ä—å –ø–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç ¬´–≤ –¥–æ–º–∏–∫–µ¬ª (–æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω). –ü–µ—Ä–µ—Ö–æ–¥–∏–º –∫–æ –≤—Ç–æ—Ä–æ–º—É –∏ –ø–æ–≤—Ç–æ—Ä—è–µ–º –ø—Ä–æ—Ü–µ—Å—Å –¥–ª—è –æ—Å—Ç–∞–≤—à–µ–≥–æ—Å—è —Ö–≤–æ—Å—Ç–∞.

```python
	for j in range(i+1, N): # –ø–æ–∏—Å–∫ –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è —Å—Ä–µ–¥–∏ –æ—Å—Ç–∞–≤—à–∏—Ö—Å—è —ç–ª.
		if m > a[j]:
			m = a[j]
			p = j
	
	if p != i: # –æ–±–º–µ–Ω –∑–Ω–∞—á–µ–Ω–∏—è–º–∏, –µ—Å–ª–∏ –±–≤–ª –Ω–∞–π–¥–µ–Ω –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π –Ω–µ –≤ i-–ø–æ–∑–∏—Ü–∏–∏
		t = a[i]
		a[i] = a[p]
		p = t

print(a)
```
