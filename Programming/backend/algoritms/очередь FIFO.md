Подход FIFO (Первый зашел — Первый вышел).Это классическая **Очередь**. Кто первым встал, того и тапки.

**Где используется:**
- **Принтер:** Документы печатаются в том порядке, в котором их отправили.
- **Очередь заявок на сервере:** Когда тысячи пользователей заходят на сайт, сервер обрабатывает их запросы по очереди.
- **Переменные в сетевых пакетах:** Данные передаются потоком; первый бит, который был отправлен, первым и будет принят.
- **Алгоритм BFS (Поиск в ширину):** Используется в графах и навигаторах для поиска кратчайшего пути.

Реализация Очереди (FIFO) на Python

Для очереди лучше не использовать обычный список (это медленно), а использовать специальный модуль `collections.deque`:
```python
from collections import deque

# Создаем очередь
queue = deque(["Клиент 1", "Клиент 2", "Клиент 3"])

# Добавляем в конец
queue.append("Клиент 4")

# ПЕРВЫЙ зашел — ПЕРВЫЙ вышел (popleft)
first_served = queue.popleft()

print(f"Обслужили: {first_served}") # Клиент 1
print(f"Остались в очереди: {list(queue)}") # Клиент 2, 3, 4
```
