**LIFO** (Last-In, First-Out) — это принцип «последним пришёл — первым ушёл»

Представь стопку тарелок: ты кладешь новую сверху, и чтобы взять тарелку, ты тоже берешь самую верхнюю (ту, которую положил последней).

В программировании реализация этого принципа называется **Стек (Stack)**

==Основные операции:==
1. **Push** — добавить элемент на вершину стека.
2. **Pop** — удалить и вернуть элемент с вершины.
3. **Peek (или Top)** — посмотреть верхний элемент, не удаляя его.
4. **isEmpty** — проверить, не пуст ли стек.

==Пример реализации стека, на примере проверки в закрывающихся скобок (")", "}", "]") если они есть в исходной строке==
```python
a = int("Введите строку: ")  
stack = []  
flVerify = True  
  
for lt in a:  
    if lt in "({[":  
        stack.append(lt)  
    elif lt in ")]}":  
        if len(stack) == 0:  
            flVerify = False  
            break  
        last = stack.pop()  
        if last == "(" and lt == ')':  
            continue  
        if last == "[" and lt == ']':  
            continue  
        if last == "{" and lt == '}':  
            continue  
  
        flVerify = False  
        break  
if flVerify and len(stack) == 0: # стек по итогам проверки должен быть пустым  
    print("yes")  
else:  
    print("no")
```

==Где это используется в реальной жизни:==
- **Кнопка «Назад» в браузере:** последняя открытая страница — первая, на которую ты вернешься.
- **Отмена действия (Ctrl+Z):** откатывается самое последнее изменение.
- **Рекурсия:** когда функция вызывает сама себя, система использует «стек вызовов» (Call Stack), чтобы помнить, куда вернуться после завершения функции.
- **Парсинг скобок:** проверка правильности расстановки скобок в коде (например, в математических выражениях).
