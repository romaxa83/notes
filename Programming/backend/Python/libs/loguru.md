##### üî∏ –î–µ–∫–æ—Ä–∞—Ç–æ—Ä¬†`@logger.catch`

–ú–æ–∂–Ω–æ –ø—Ä–æ—Å—Ç–æ "–æ–±–µ—Ä–Ω—É—Ç—å" –≤ –Ω–µ–≥–æ –ª—é–±—É—é —Ñ—É–Ω–∫—Ü–∏—é, –∏ Loguru –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –ø–æ–π–º–∞–µ—Ç –ª—é–±–æ–µ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –≤ –Ω–µ–π –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç, –∏ –≤—ã–≤–µ–¥–µ—Ç —Å–∞–º—ã–π –ø–æ–¥—Ä–æ–±–Ω—ã–π –æ—Ç—á–µ—Ç –∏–∑ –≤–æ–∑–º–æ–∂–Ω—ã—Ö.

##### üî∏ –ö–∞—Å—Ç–æ–º–∏–∑–∞—Ü–∏—è —Ñ–æ—Ä–º–∞—Ç–∞ –ª–æ–≥–æ–≤

–ú–æ–∂–Ω–æ —Å–¥–µ–ª–∞—Ç—å —Å –ø–æ–º–æ—â—å—é –ø–∞—Ä–∞–º–µ—Ç—Ä–∞¬†`format`¬†–≤¬†`logger.add()`. –ù–æ –∑–¥–µ—Å—å –∫—Ä–æ–µ—Ç—Å—è –≤–∞–∂–Ω—ã–π –Ω—é–∞–Ω—Å, —Å–≤—è–∑–∞–Ω–Ω—ã–π —Å —Ü–≤–µ—Ç–∞–º–∏.
–ú–∞–≥–∏—è —Ü–≤–µ—Ç–∞ Loguru —Ä–∞–±–æ—Ç–∞–µ—Ç –±–ª–∞–≥–æ–¥–∞—Ä—è —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–º —Ç–µ–≥–∞–º, –ø–æ—Ö–æ–∂–∏–º –Ω–∞ HTML, –∫–æ—Ç–æ—Ä—ã–µ –≤—Å—Ç—Ä–æ–µ–Ω—ã –≤ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π —Ñ–æ—Ä–º–∞—Ç. –ì–ª–∞–≤–Ω—ã–π –∏–∑ –Ω–∏—Ö ‚Äî —ç—Ç–æ —Ç–µ–≥¬†`<level>`. –ò–º–µ–Ω–Ω–æ –æ–Ω –≥–æ–≤–æ—Ä–∏—Ç Loguru —Ä–∞—Å–∫—Ä–∞—Å–∏—Ç—å —Ç–µ–∫—Å—Ç –≤ —Ü–≤–µ—Ç, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–π —É—Ä–æ–≤–Ω—é —Å–æ–æ–±—â–µ–Ω–∏—è.

==–î–∞–≤–∞–π—Ç–µ –ø–æ—Å–º–æ—Ç—Ä–∏–º, —á—Ç–æ –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç, –µ—Å–ª–∏ –º—ã –∑–∞–¥–∞–¥–∏–º –ø—Ä–æ—Å—Ç–æ–π —Ñ–æ—Ä–º–∞—Ç –±–µ–∑ —ç—Ç–∏—Ö —Ç–µ–≥–æ–≤:==

==–ß—Ç–æ–±—ã –≤–µ—Ä–Ω—É—Ç—å "—É–º–Ω–æ–µ" —Ä–∞—Å–∫—Ä–∞—à–∏–≤–∞–Ω–∏–µ, –Ω—É–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç–µ–≥–∏ –≤ —Å—Ç—Ä–æ–∫–µ —Ñ–æ—Ä–º–∞—Ç–∞==
