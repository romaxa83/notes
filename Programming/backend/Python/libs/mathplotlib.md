##### üî∏ –†–∞—Å—à–∏—Ä–µ–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª–∞ —Å matplotlib

–î–æ–ø—É—Å—Ç–∏–º, –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø–æ—Å—Ç—Ä–æ–∏—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –¥–∏–∞–≥—Ä–∞–º–º –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ —Å –ø–æ–º–æ—â—å—é seaborn, –≤ —ç—Ç–æ–º —Å–ª—É—á–∞–µ –≤—ã –º–æ–∂–µ—Ç–µ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è —Ñ—É–Ω–∫—Ü–∏–µ–π¬†`subplot`¬†–∏–∑ matplotlib.–° –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏¬†`subplot`¬†–Ω–∞ –æ–¥–Ω–æ–º –≥—Ä–∞—Ñ–∏–∫–µ –º–æ–∂–Ω–æ –ø–æ—Å—Ç—Ä–æ–∏—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –¥–∏–∞–≥—Ä–∞–º–º. –§—É–Ω–∫—Ü–∏—è –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –≤ —Å–µ–±—è —Ç—Ä–∏ –ø–∞—Ä–∞–º–µ—Ç—Ä–∞: –ø–µ—Ä–≤—ã–π ‚Äì –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å—Ç—Ä–æ–∫, –≤—Ç–æ—Ä–æ–π ‚Äì –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å—Ç–æ–ª–±—Ü–æ–≤, —Ç—Ä–µ—Ç–∏–π ‚Äì –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –¥–∏–∞–≥—Ä–∞–º–º.

##### üî∏ –í–∞—Ä–∏–∞–Ω—Ç—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è

–ø—Ä–∏–º–µ—Ä—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –∏–∑ –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏–∏ - https://seaborn.pydata.org/examples/index.html

```python
tips_df = sns.load_dataset('tips')
tips_df.head()
tips_df["tip_percentage"] = tips_df["tip"] / tips_df["total_bill"]

# –ü—Ä–æ—Ü–µ–Ω—Ç —á–∞–µ–≤—ã—Ö
sns.histplot(tips_df["tip_percentage"], binwidth=0.05)

# –¢–∞–∫–∂–µ –º–Ω–µ –±—ã–ª–æ –∏–Ω—Ç–µ—Ä–µ—Å–Ω–æ, –º–µ–Ω—è–µ—Ç—Å—è –ª–∏ –ø—Ä–æ—Ü–µ–Ω—Ç —á–∞–µ–≤—ã—Ö –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –≤—Ä–µ–º–µ–Ω–∏ –¥–Ω—è
sns.histplot(data=tips_df, x="tip_percentage", binwidth=0.05, hue="time")

# –û–±—â–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —á–∞–µ–≤—ã—Ö –∑–∞ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –¥–µ–Ω—å –Ω–µ–¥–µ–ª–∏
sns.barplot(data=tips_df, x="day", y="tip", estimator=np.sum)

# –í–ª–∏—è–Ω–∏–µ —Ä–∞–∑–º–µ—Ä–∞ —Å—Ç–æ–ª–∏–∫–∞ –∏ –¥–Ω—è –Ω–µ–¥–µ–ª–∏ –Ω–∞ —á–∞–µ–≤—ã–µ
pivot = tips_df.pivot_table(
    index=["day"],
    columns=["size"],
    values="tip_percentage",
    aggfunc=np.average)
sns.heatmap(pivot)
```
