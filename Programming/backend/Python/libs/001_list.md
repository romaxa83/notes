#### üîπ `Schedule` ‚Äî –ø–ª–∞–Ω–∏—Ä–æ–≤—â–∏–∫

**–ö–∞–∫—É—é –±–æ–ª—å —Ä–µ—à–∞–µ—Ç:**¬†–°–ª–æ–∂–Ω–æ—Å—Ç—å –Ω–∞—Å—Ç—Ä–æ–π–∫–∏ –ø–µ—Ä–∏–æ–¥–∏—á–µ—Å–∫–∏—Ö –∑–∞–¥–∞—á –≤–Ω—É—Ç—Ä–∏ —Å–∫—Ä–∏–ø—Ç–∞.¬†`Schedule`¬†–Ω–µ —Ç—Ä–µ–±—É–µ—Ç –Ω–∞—Å—Ç—Ä–æ–π–∫–∏ –¥–µ–º–æ–Ω–æ–≤, –æ–Ω —Ä–∞–±–æ—Ç–∞–µ—Ç –≤–Ω—É—Ç—Ä–∏ –≤–∞—à–µ–≥–æ Python-–ø—Ä–æ—Ü–µ—Å—Å–∞.

```python

import time

def job():
    print("–î–µ–ª–∞—é –±—ç–∫–∞–ø –±–∞–∑—ã –¥–∞–Ω–Ω—ã—Ö...")

# –°–∏–Ω—Ç–∞–∫—Å–∏—Å, –∫–æ—Ç–æ—Ä—ã–π –ø–æ–π–º–µ—Ç –¥–∞–∂–µ –≤–∞—à–∞ –±–∞–±—É—à–∫–∞

schedule.every().wednesday.at("13:15").do(job)

while True:
    schedule.run_pending()
    time.sleep(1)
```
