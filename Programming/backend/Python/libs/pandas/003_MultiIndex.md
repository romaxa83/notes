MultiIndex ‚Äî —ç—Ç–æ –æ–±—ä–µ–∫—Ç –¥–ª—è –∏–Ω–¥–µ–∫—Å–∞, –≤–∫–ª—é—á–∞—é—â–∏–π –Ω–µ—Å–∫–æ–ª—å–∫–æ —É—Ä–æ–≤–Ω–µ–π. –ö–∞–∂–¥—ã–π —É—Ä–æ–≤–µ–Ω—å —Å–æ–¥–µ—Ä–∂–∏—Ç –∑–Ω–∞—á–µ–Ω–∏–µ –∏–Ω–¥–µ–∫—Å–∞ –¥–ª—è —Å—Ç—Ä–æ–∫–∏. MultiIndex –Ω–∞–∏–±–æ–ª–µ–µ —É–¥–æ–±–µ–Ω, –∫–æ–≥–¥–∞ –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä –¥–ª—è —Å—Ç—Ä–æ–∫–∏ –¥–∞–Ω–Ω—ã—Ö —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ –Ω–µ–∫–æ–µ–≥–æ —Å–æ—á–µ—Ç–∞–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏–π.
